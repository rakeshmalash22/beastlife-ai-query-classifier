# Beastlife AI Customer Query Classifier

An AI-powered customer support automation prototype built using **Python, Machine Learning, and Streamlit**.

This system automatically analyzes customer queries and classifies them into different issue categories. It also visualizes the distribution of customer issues through an interactive dashboard.

---

## Project Objective

The goal of this project is to design a prototype system that can:

* Automatically analyze incoming customer queries
* Classify problems into categories using AI
* Visualize issue distribution in a dashboard
* Help companies identify the most common customer problems

---

## Example Issue Categories

The system can classify queries into categories such as:

* Order Tracking
* Delivery Delay
* Refund Request
* Product Issue
* Payment Issue
* General Query

Example:

Customer Query:

Where is my order?

Detected Category:

Order Tracking

### 2 Run the application

### 3 Open in browser
